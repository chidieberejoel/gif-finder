# gif-finder

## Installation and Set up
Node.js should be installed. Get Node.js [here](https://nodejs.org/en/download/).

Navigate to a directory of your choice using a CLI and run the following commands in the order:

```
git clone https://github.com/chidieberejoel/gif-finder.git
cd gif-finder
```
## Run the application
Instructions on how to run the [Client](./client) and [Server](./server) sides of the application can be found in their respective folders.
